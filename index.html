CSS Snippet Cheatsheet

Here are some commonly used css snippets

They are gathered here for future reference

Flexbox Row

Use these properties to create a Flexbox row layout with wrapping

.row {
    display: flex;
    flex-direction: row;
    flex-wrap: wrap;
}

Flexbox Column

Use these properties to create a Flexbox column layout

.column {
    display: flex;
    flex-direction: column;
}

CSS Grid Layout

Build an even six column layout using CSS Grid

.grid {
    display: grid;
    width: 100%;
    grid-template-columns: 1fr 1fr 1fr 1fr 1fr 1fr;
}

Linear Gradients

Use these properties to create a background linear gradient from top to bottom

.linear-gradient-background {
    background-image: linear-gradient(
        rgba(160, 40, 230, 0.5) 0%,
        rgba(232, 156, 230, 0.8) 100%
    );
}

Box Transition Glow

Use transition and box shadows to create a glow effect on hover

.code-card .card-header {
    border-radius:8px;
    transition: all 0.5s ease-in-out;
}

.code-card:hover, 
.code-card:hover .card-header{
    box-shadow: 
      inset 0px 0px 8px rgba(232, 102,236, 1),
            0px 0px 15px rgba(232, 102,236, 1);
}

Overlay Card with Title

Use position properties and negative margins to raise elements higher than their natural starting point

.card-header {
    position: relative
    margin-top: -16px;
}
